## üïπÔ∏è How to Play

### 2-Player Mode:
- **Player 1**:
  - Movement: `W`, `A`, `S`, `D`
  - Light Attack: `J`
  - Strong Attack: `I`
- **Player 2** (Controller):
  - Movement: D-Pad and A to jump
  - Attacks: B and Y

### Solo Mode:
- Use the **Arrow Keys** to control Player 1
- Press **Spacebar** to auto-select a character for Player 2
